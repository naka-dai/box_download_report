Box Daily Update - 図面活用状況レポート 自動更新パッケージ
================================================================================

■ このパッケージについて

Box APIから図面フォルダのアクセスログを取得し、
運用開始前後の比較分析ができるダッシュボードを自動生成します。

タスクスケジューラで毎日実行することで、常に最新のデータを表示できます。


★★★ すぐに使えます！ ★★★

このパッケージは完全に設定済みです:
・config.json（Box認証ファイル）同梱
・.env（設定ファイル）事前設定済み
・BOX_ROOT_FOLDER_ID 設定済み

→ フォルダをコピーして、すぐに実行できます！


■ 必要なもの

1. Windows 10以降のPC
2. インターネット接続（Box APIアクセス用）

以上！ Pythonのインストールも不要です。


■ クイックスタート（3ステップ）

【1】フォルダをコピー
  このフォルダ（box_daily_update）を任意の場所にコピー
  例: C:\BoxReports\box_daily_update\

【2】実行
  コマンドプロンプトで:
  1. cd C:\BoxReports\box_daily_update
  2. box_daily_update.exe

【3】確認
  data\dashboard_period_allinone_full.html をブラウザで開く

→ これだけで完了！

【4】タスクスケジューラに登録（毎日自動実行・オプション）
  詳細は「セットアップ手順.txt」を参照


■ 出力されるファイル

data/dashboard_period_allinone_full.html
  → 図面活用状況ダッシュボード
  → ブラウザで開いて閲覧
  → 期間フィルター: 全期間/運用開始前(～2025-10-13)/運用開始後(2025-10-14～)
  → 3つのタブ: 統合レポート/ダウンロードのみ集計/プレビューのみ集計
  → マウスオーバーでユーザー詳細表示

data/box_audit.db
  → SQLiteデータベース（累積データ）
  → 削除しないこと


■ フォルダ構成

box_daily_update/
├── box_daily_update.exe              ← 実行ファイル（7.5 MB）
├── config.json                        ← Box認証ファイル（同梱済み）★
├── .env                               ← 設定ファイル（事前設定済み）★
├── .env.sample                        ← 設定サンプル

├── README.txt                         ← このファイル
└── セットアップ手順.txt              ← 詳細手順
